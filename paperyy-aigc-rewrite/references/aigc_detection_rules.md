# PaperYY AIGC检测规律参考手册

> 基于paperyy平台8份AIGC检测报告的系统性分析整理
> 适用于教育科研论文（准实验研究/教学实证类）

---

## 一、系统检测机制

### 1.1 三级判定标准

| 颜色 | 概率范围 | 判定 |
|-----|--------|-----|
| 黑色 | 0%–40% | 低度疑似（人类创作） |
| 橙色 | 40%–60% | 中度疑似 |
| 红色 | 60%–100% | 高度疑似（AI创作） |
| 灰色 | — | 不予检测（标题、公式、图表、参考文献等） |

### 1.2 段落粒度检测

- 系统对每个独立段落单独评分
- 段落内的AI特征会拉高整段得分，而非平均全文
- 概率分布两极化：引言/综述/结论（90%+）vs 方法细节/叙事（<20%）

### 1.3 字数与疑似率负相关

| 字数区间 | 平均疑似率 | 原因 |
|--------|---------|------|
| 3,000–4,500字 | ~57% | 高疑似段占比高 |
| 4,500–6,000字 | ~60% | 扩写多为综述（仍高风险） |
| 7,500字以上 | ~43% | 增加了叙事性内容（低风险） |

### 1.4 不予检测内容

标题、图注、公式、表格、参考文献均不计入检测。
**策略：** 增加图表注释、步骤分项、小标题等"不予检测"内容可稀释疑似率。

## 二、必命中句型清单（98%+判定为AI）

| 标签 | 典型句式 | 触发概率 |
|-----|---------|---------|
| A-政策引用开篇 | "《XXX纲要》明确提出/强调……" | 96.9% |
| B-演进叙事 | "已从……向……演进/转变/跨越" | 85~96.9% |
| C-统计结论 | "实验组……显著高于对照组（p<0.001）" | 96.9% |
| D-概念定义 | "XXX是指以……方式……的过程/素养" | 94~96.9% |
| E-研究缺口 | "现有研究多聚焦……缺乏……本研究创新在于" | 92.9~96.9% |
| F-结论三段式 | "首先……其次……最后……" | 94.5~96.9% |

## 三、高风险段落结构清单

### 3.1 引言四段式框架（每段均高危）
```
第1段：政策背景（文件引用）→ 极高
第2段：现实困境（并列问题）→ 高
第3段：技术解决路径 → 极高
第4段：研究目的（本研究旨在）→ 高
```

### 3.2 文献综述三段式
```
2.1 概念定义（定义+理论演变+测量工具）→ 极高
2.2 应用现状（演进叙事+两条路线+文献引用）→ 极高
2.3 研究缺口（现有不足+本研究填补）→ 极高
```

### 3.3 研究方法中的雷区
- "非对等控制组前后测设计"（12.7%~94.5%，取决于有无个人叙述）
- "自变量为……因变量为……"（术语密集段）
- "参考……测验，开发……评分者一致性信度系数为0.89"

### 3.4 结论讨论中的高危结构
- "本研究与既有研究的呼应"（"与XXX的判断形成了呼应"）
- 实践建议三条目（"第一……第二……第三……"）
- 未来展望三方向（"首先……其次……最后……"）

## 五、低风险内容特征（<40%疑似率）

- **具体人名/地名/工具名**："人教版""NB化学虚拟实验室""某市两所公办学校"
- **口语化转折**："说它好看，是因为……说它不好上，是因为……"
- **感受性描述**："这一点在教学实践中感触很深""教学实践表明"
- **学生原话引用**："做错了也不会挨批评""可以随便试，大不了重来"
- **具体数字+场景**："每周仅安排一节实验课""4人一组"
- **诚恳局限表达**："120名学生放到整个初中化学教学的背景下，规模很有限"
- **操作过程场景描写**："给学生整整一节课的时间，在NB虚拟实验室里自行设计方案"

## 六、有效降分策略

### 策略一：替换政策套语为具体案例
- ✗ "《教育强国建设规划纲要》明确提出推进教育数字化……"（96.9%）
- ✓ "以人教版九年级化学'实验室制取氧气'为例，从原理讲解到学生分组实验，至少需要两课时……"（49.9%）

### 策略二：场景化过程叙述替换学术化论断
- ✗ "AI赋能的实验环境为学生提供了低成本的试错空间与个性化引导"（95.5%）
- ✓ "学生在虚拟平台上对同一个实验先后尝试了五六种方案，这种迭代过程在传统实验室里是不可想象的"（12.7%）

### 策略三：加入学生原话/访谈直接引语
效果：12.7%~30%。必须真实感强，非转述式总结。

### 策略四：研究局限坦诚表达
效果：~27%。诚恳承认不足的语气模式不符合AI写作倾向。

### 策略五：讨论章节加入一线教师体会
效果：<20%。第一人称感受+具体教学场景描述。

### 策略六：增加叙事性内容（非综述性）
扩写必须扩充叙事性内容而非综述性内容，否则疑似率不降反升。

## 七、各章节改写对照模板

### 摘要
| 原版特征 | 改写方向 |
|---------|---------|
| 技术演进套语+研究意义升华 | 从"课堂困扰"切入，口语化 |
| 统计结论标准句"极显著优于" | 改为"差异明显/大幅领先"的叙事性描述 |
| "为…提供实证支持/范式参照"收尾 | 改为具体建议或开放性问题 |

### 英文摘要
| 原版特征 | 改写方向 |
|---------|---------|
| "With the deep integration of…" | 第一人称叙事，从教学现实切入 |
| 标准准实验描述句 | 叙事性实验描述 |
| "The results showed that…" | "The difference in outcomes was noticeable." |

### 引言
| 原版特征 | 改写方向 |
|---------|---------|
| 政策文件引用开篇 | 从具体课例/教学困境切入 |
| "一方面……另一方面……"并列枚举 | 改为散文化叙述 |
| 政策内容单独成段 | 夹叙夹引融入叙事 |

### 文献综述
| 原版特征 | 改写方向 |
|---------|---------|
| "研究表明/证实/揭示"演进叙事 | 疑问句+第一人称阅读视角 |
| "XXX是指……的过程/素养"定义式 | 对话式进入，承认定义不统一 |
| "现有研究多聚焦……缺乏……" | 具体列举空白，避免"填补空白"表述 |

### 研究方法
| 原版特征 | 改写方向 |
|---------|---------|
| "本研究采用准实验研究中的非对等控制组前后测设计" | 叙事性决策陈述+选择原因 |
| "自变量为……因变量为……" | 融入叙事中描述 |
| 研究工具仅介绍功能 | 加入开发过程/测试经历的叙述 |

### 结果分析
| 原版特征 | 改写方向 |
|---------|---------|
| 统计结论句"极显著高于对照组" | 数据保持，解读段加入教师观察视角 |
| 表格引用后紧跟标准分析 | 用叙事性方式讲述数据背后的"故事" |

### 讨论
| 原版特征 | 改写方向 |
|---------|---------|
| "第一……第二……第三……"三条路径 | 故事化叙述，去掉序号结构 |
| "本研究与XXX的判断形成了呼应" | 具体对比差异与共同点 |
| 实践建议三条目 | 分段标题式（"关于教师角色/关于虚实节奏"） |

### 结论与展望
| 原版特征 | 改写方向 |
|---------|---------|
| 三方向展望"首先…其次…最后…" | 开放性追问+具体下一步计划 |
| "为…提供有益参照" | 务实的个人感悟+承认局限 |

## 八、顽固高风险区（难以低于80%）

| 位置 | 最低记录 | 原因 |
|-----|---------|------|
| 中文摘要 | 93.7% | 摘要格式本身高度规范化 |
| 英文摘要结论句 | 96.9% | 英文学术摘要套路 |
| 文献综述2.1（概念定义） | 93.7% | 定义式写法 |
| 研究缺口2.3 | 92.9% | 套路化创新声明 |
| 统计结论句 | 96.9% | 数据描述格式固定 |

**应对策略：** 接受这些位置的高疑似率，通过扩充低风险叙事性内容（讨论、访谈分析、操作描述）来拉低整体疑似率至15%左右。
