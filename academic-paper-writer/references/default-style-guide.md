# 通用学术论文排版规范（默认）

## 适用范围

当用户未提供目标期刊/会议的具体格式要求时，使用本规范作为默认排版标准。
本规范综合了中文社科期刊、中文学术年会报告集的通用惯例。

---

## 页面设置

- 纸张：A4
- 页边距：上下 2.54cm，左右 3.18cm
- 行距：1.5 倍行距
- 页码：底部居中

---

## 字体规范

| 元素 | 字体 | 字号 | 格式 |
|------|------|------|------|
| 论文主标题 | 黑体 | 小二（18pt） | 居中，加粗 |
| 副标题 | 楷体 | 小三（15pt） | 居中 |
| 作者姓名 | 宋体 | 四号（14pt） | 居中 |
| 作者单位 | 楷体 | 小五（9pt） | 居中，加括号 |
| "摘 要"标签 | 黑体 | 五号（10.5pt） | 加粗 |
| 摘要正文 | 楷体 | 五号（10.5pt） | 首行缩进2字符 |
| "关键词"标签 | 黑体 | 五号（10.5pt） | 加粗 |
| 关键词内容 | 楷体 | 五号（10.5pt） | — |
| 作者简介 | 宋体 | 小五（9pt） | — |
| 一级标题 | 黑体 | 四号（14pt） | 加粗 |
| 二级标题 | 黑体/宋体 | 小四（12pt） | 加粗 |
| 三级标题 | 宋体 | 五号（10.5pt） | 加粗 |
| 正文 | 宋体 | 五号（10.5pt） | 首行缩进2字符 |
| 图题 | 宋体 | 小五（9pt） | 居中，图下方 |
| 表题 | 黑体 | 小五（9pt） | 居中，表上方，加粗 |
| 参考文献 | 宋体 | 小五（9pt） | 左对齐，悬挂缩进 |

---

## 标题格式

- 一级标题段前段后各 0.5 行
- 二级标题段前 0.3 行，段后 0
- 各节之间正文不另起页（结论和参考文献可另起页）
- 标题不加句号

---

## 作者简介格式

```
作者简介：姓名（出生年月），性别，籍贯（省+市/县），学历，工作单位全称，职称/职务，
主要研究方向：XXX。
```

多作者合著时分行排列，通讯作者标明 * 号。

---

## 基金项目格式

```
基金项目：{项目名称}（项目编号：XXXXXX）
```

---

## 英文摘要格式（如需）

放在中文关键词之后，格式：
```
Abstract: [英文摘要内容]
Keywords: keyword1; keyword2; keyword3
```

---

## 图表规范

### 图
- 图题格式：`图1  {图题名称}`（图号与图题之间空2格）
- 位置：图题在图下方，居中
- 图注（如有）：在图题下方，五号字，左对齐

### 表
- 表题格式：`表1  {表题名称}`（表号与表题之间空2格）
- 位置：表题在表上方，居中
- 三线表：顶线和底线用粗线（1.5pt），中间横线用细线（0.75pt）
- 表注（如有）：在表下方，小五号字

### 先文后图表
- 正文中写"如图1所示""见表1"，再紧接放出图表
- 图表编号全文连续（图1、图2...，表1、表2...）
- 图表中的数字和英文使用 Times New Roman 字体

---

## 参考文献列表格式

- 参考文献前加粗标注"参考文献"，单独一行
- 按正文首次引用顺序编号 [1][2]...
- 悬挂缩进：第一行不缩进，续行缩进2字符
- 文献间不加空行
